# Assets

## Musique de fond

Placez votre fichier de musique romantique ici et nommez-le `music.mp3`.

### Suggestions de musique libre de droits :
- Une musique douce et romantique au piano
- Une mélodie de boîte à musique
- Une chanson instrumentale légère

### Sources de musique libre de droits :
- [Pixabay Music](https://pixabay.com/music/)
- [Free Music Archive](https://freemusicarchive.org/)
- [Bensound](https://www.bensound.com/)

Le fichier doit être au format MP3 pour une meilleure compatibilité.
